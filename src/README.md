<h1>Push Swap Planning</h1>

Push Swap é um programa que ordena uma determinada [pilha](https://en.wikipedia.org/wiki/Stack_(abstract_data_type)) de dados. É simplesmente isso que o programa deve fazer.

<h3>Detalhamento:</h3>
<ul>
	<li>O Programa deve receber um agumento (lista de números)</li>
	<li>O Programa deve exibir uma menssagem de erro caso o argumento seja inválido</li>
	<li>O Resultado do programa deve ser uma lista de intruções que servirão para ordenar a lista.</li>
	<li>Cada instrução deve ser acompanhada de um '\n' e nada mais.</li>
</ul>
